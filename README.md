# 간단한 게시판 프로젝트
> Board는 스프링 프레임워크를 학습하면서 간단한 게시판을 개발하는 프로젝트이다.

## 개발 환경 설정
* Spring Framework 4.3.16
* Mysql 5.7.21
* Spring Tool Suite 3.9.1.RELEASE
* Mybatis 3.4.5
